      Access to Hamilton #

      For many of the exercises in the course, we will be using the Hamilton supercomputer. You should have already obtained an account on Hamilton as part of Core Ib. If you do not yet have one, please register following their instructions.

      Since we’ll be logging in a lot, I also provide some tips on how to configure ssh for swifter login.

      Quickstart reminder #

      If you want to remind yourself of how Hamilton works, take a look at the quickstart guide from PHYS52105.
